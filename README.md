# HellVantage: Offensive Cloud SAST

HellVantage is an elite Static Application Security Testing (SAST) engine designed for modern Cloud Infrastructure as Code (IaC).

## Why HellVantage?
- Offensive DNA: Rules engineered from real-world exploit paths.
- Multi-Cloud Ready: Deep inspection for Terraform and CloudFormation.
- Enterprise Reporting: Native SARIF output integration.

## Quick Start
Uses the following in your GitHub Action:
- uses: your-org/hellvantage@v1
  with:
    target_directory: '.'
    output_format: 'sarif'
